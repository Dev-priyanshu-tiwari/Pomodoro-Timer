# ⏱️ FlowState — Pomodoro Timer

A dark, sleek Pomodoro productivity timer built with **HTML, CSS & Vanilla JavaScript**. No frameworks, no libraries — pure frontend engineering.

> Lets Used Time effectively

Test NOW -  https://dev-priyanshu-tiwari.github.io/Pomodoro-Timer/

---

## ✨ Features

- ⏱️ **Pomodoro Timer** — Focus, Short Break, Long Break modes
- 🔵 **Animated SVG Progress Ring** — glows in mode-specific colors
- 🌿 **Session Dots** — track completed pomodoros in a cycle
- ✅ **Task Manager** — add, complete, delete, and set active tasks
- ⚙️ **Customizable Settings** — durations, sessions, auto-start toggles
- 🔔 **Web Audio API Sounds** — synthesized tones, no audio files needed
- 📊 **Daily Stats** — pomodoros, focus time, streak (resets daily)
- 💾 **LocalStorage** — persists tasks, config, and stats across sessions
- 🎨 **Color theme** shifts per mode (violet → emerald → blue)
- 📱 Fully responsive on all devices

## 🎯 How to Use

1. Click **START** to begin a 25-minute focus session
2. Add tasks in the task input and click them to set as active
3. When the timer ends, you'll hear a sound and auto-advance to break
4. After 4 pomodoros, a long break is earned
5. Customize everything in **⚙ Settings**
